# Brain Tumor Classification using VGG16 Transfer Learning

## Overview
Brain MRI classification project using VGG16 Transfer Learning.

Classes:
- Glioma
- Meningioma
- Pituitary
- No Tumor

## Technologies
- Python
- TensorFlow
- Keras
- NumPy
- Scikit-Learn
- Matplotlib

## Run
```bash
pip install -r requirements.txt
python src/train.py
```
